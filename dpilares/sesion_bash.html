Introducción a bash

Bash: Es un lenguaje de comandos y una UNIX shell. Acrónimo de: Bourne-Again SHell..

Shell: Intérprete de comandos, permite la ejecución interactiva y no interactiva de comandos.

Scripting: Ejecución de comandos automática.

Script: 1-tareas.sh

No olvides el permiso de ejecución

Nuestro intérprete de comandos

Para saber qué shell estamos ocupando


					echo $SHELL

Para saber dónde está nuestro ejecutable de la shell


						which bash

Para saber qué otras shells tenemos disponibles


						cat /etc/shells

						#!/bin/bash

stdout:standard output. La salida de los comandos. Se redirige con >

stderr:standard error. La salida de los errores. Se redirige con 2>

Ambas se redirigen con &> 


						ls barfoo > stdout.txt
						ls barfoo &> stdoutandstderr.txt

Comparación de números y cadenas de caracteres

Descripción Números Strings
menor que -lt <
mayor que -gt >
igual -eq =
no igual -ne !=
igual o menor -le N/A
igual o mayor -ge N/A

Ejemplos

Script: 8-comparaciones.sh


						a=3
						b=8
						[ $a -lt $b ]
						echo $?
						[ $a -ge $b ]
						echo $?
						[ $a -eq $b ]
						echo $?

0 es verdadero, 1 es falso

$? nos da el último status de salida.

Ejemplos


						[ "naranjas" = "manzanas" ]
						echo $?
						str1="uvas"
						str2="uvas"
						[ $str1 = $str2 ]
						echo $?

0 es verdadero, 1 es falso

Ciclos

Son sentencias que nos permiten iterar o ejecutar varias veces alguna tarea hasta que todos los elementos en una lista se cumplen o hasta que se cumplen ciertas condiciones predefinidas.

for: Se utiliza para iterar en un número de elementos en una lista


						 for i in 1 2 3 4 5; do echo $i; done
					

						for i in $( cat items.txt ); do echo -n $i | wc -c; done

Archivo: items.txt

Operaciones matemáticas en la terminal

siempre con doble paréntesis 


						x=15;y=8
						z=$(( $x * $y ))
						echo $z

o con el comando expr


						expr 100 - 80
						expr 6 \* 8

o con el comando let, que guarda el resultado en una variable


						let a=2+2
						echo $a
						let b=3*($a-1)
						echo $b
						let c=($b**2)/2
						echo $c
						let c++
						let c--

Script: 16-backup-let.sh

Finalmente, tenemos el comando bc, que es toda una calculadora

						echo $x
						echo 'scale=30;8.5 / 2.3' | bc
