Bootstrap

¿Qué es Bootstrap?

  • Es el framework de front-end más popular
  • Se utiliza para construir sitios web y aplicaciones mobile first
    • Las reglas de CSS por default aplican a los dispositivos móviles.
  • Utiliza HTML, CSS y JavaScript

¿Por qué se utiliza?

  • Reduce el tiempo de desarrollo.
  • Asegura que el diseño sea responsivo.
  • No tienes que repetir muchas veces lo mismo en diferentes proyectos.
  • Es compatible en todos los navegadores.
  • Se puede modificar.

Tiene muchísimos componentes

  • Alertas
  • Botones
  • Cards
  • Carrusel (slideshow)
  • Dropdowns
  • Spinners
  • Tooltips
  • Ventanas modales
  • Barras de navegación
  • Páneles
  • Barras de progreso
  • Scrollspy
  • Íconos
  • Tablas

Breakpoints y media queries

Nos sirven para especificar reglas de css que van a aplicarse solamente a tamaños específicos del viewport. En Bootstrap:

Breakpoint Class infix Dimensions
X-Small None <576px
Small sm ≥576px
Medium md ≥768px
Large lg ≥992px
Extra large xl ≥1200px
Extra extra large xxl ≥1400px

Contenedores

Los contenedores son los bloques fundamentales para alinear, contenido en Bootstrap y se necesitan para que funcione su grid system. Hay tres tipos de clases para manejar contenedores:

  • .container: tiene ancho fijo y cambia su max-width en cada breakpoint.
  • .container-{breakpoint}: su ancho es 100% hasta que se llega al breakpoint específico.
    • .container-sm: tiene 100% en pantallas de menos de 576px y después escala
  • .container-fluid: es 100% siempre.

Sistema de grid

Usa contenedores, filas y columnas para crear el layout y alinear contenido. Está construido utilizando flex.

  • Tiene 6 breakpoints reponsivos basados en media queries de min-width.
  • Los contenedores centran y ajustan el contenido.
  • Las filas permiten separar el contenido sobre la vertical y encerrar columnas.
  • El espacio entre las columnas también puede ajustarse cambiando el gutter.
