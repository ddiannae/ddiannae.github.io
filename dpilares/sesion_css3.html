Diseño responsivo

El contenido de la página cambia su tamaño o se mueve para verse bien en dispositivos con tamaños de pantalla diferentes. Solamente se utiliza CSS y HTML para lograr el objetivo.

Viewport: es el área visible para el usuario en una página web.

Necesitamos esta etiqueta en todas las páginas para controlar cómo se despliegan en diversos dispositivos.
  • width=device-width: el ancho de la página depende de la pantalla del dispositivo.
  • initial-scale=1.0: el nivel de zoom cuando la página se carga.

El objetivo es que el usuario no tenga que hacer scroll horizontal a la página en ningún dispositivo.

¿Qué más es importante?

  1. No utilizar elementos grandes con ancho fijo. Una imagen con ancho fijo podría hacer que el scroll horizontal aparezca.
  2. No diseñes pensando en un ancho de página en particular.
  3. Utilizar media queries para ajustar el ancho de los elementos. Utilizar valores para width relativos y tener cuidado con el posicionamiento absoluto.

Grid-view: dividir la página en columnas y utilizarlas para ajustar el contenido

Usualmente se utilizan 12 columnas que dan un tamaño total de 100% y se alargan o achican según el tamaño de la ventana.

¿Qué tenemos que hacer? 


                        * {
                            box-sizing: border-box;
                          }

Para que el padding y el borde se incluyan en el ancho y alto total de los elementos.

Algo simple


                        .menu {
                            width: 25%;
                            float: left;
                        }
                        .main {
                            width: 75%;
                            float: left;
                        }

Para ajustar se utilizan los media queries. Usan la regla @media para incluir bloques de propiedades de CSS solo si alguna condición se cumple.


                        @media only screen and (max-width: 600px) {
                            body {
                              background-color: lightblue;
                            }
                          }

Ejemplo de W3Schools: Cambio de color de fondo

Se diseña primero para móviles.

Cuando la pantalla mide menos de 768px cada columna tomará todo el ancho disponible. Si la pantalla es más grande, usamos el modelo de 12 columnas. 


                        /* Para móviles: */
                        [class*="col-"] {
                          width: 100%;
                        }
                        
                        @media only screen and (min-width: 768px) {
                          /* Para escritorio */
                          .col-1 {width: 8.33%;}
                          .col-2 {width: 16.66%;}
                          .
                          .
                          .col-12 {width: 100%;}
                        }

Flexbox Layout

Una forma de acomodar los elementos en una página y estructurarlos sin tener que utilizar float y posicionamiento absoluto. Antes había cuatro modos principales:

  • En bloque, para secciones de una página.
  • Inline, para texto.
  • Como tabla, para datos de dos dimensiones.
  • Posicionar de forma explícita algún elemento.
