En general, un programa es una lista de instrucciones ejecutadas por la computadora.

En un lenguaje de programació, las instrucciones se llaman declaraciones.

Las declaraciones se componen de: valores, operadores, expresiones, palabras reservadas y comentarios.

Sintaxis: Las reglas que componen un lenguaje. Describen las declaraciones válidas.

Valores

Hay dos tipos:

  • Literales. Valores fijos

                        /* Números */
                        10
                        8.340
                        /* Strings */
                        "Hola, con comillas dobles"
                        'Hola, con comillas simples'
  • Literales. Valores fijos
  • Variables. Guardan datos que pueden cambiar

                        var x; // Declaración
                        var a,b;
                        a = 10; // Asignación
                        b = 35;
                        x = a * b;

¿Qué identificadores de variables son válidos?

Antes de 2015, para declarar variables sólamente se usaba var, ahora se utiliza let y const

La diferencia entre var y let es su scope, dónde están disponibles las variables para ser usadas.

Por ahora: utilizar const si es una variable que no va a modificarse, y let en otros casos.

Funciones

Definición


                        function productoDosNumeros(num1, num2) {
                            return num1*num2;
                        }
  • Palabra reservada function.
  • Un nombre + paréntesis.
  • Parámetros opcionales separados por comas.
  • Código a ejecutarse entre {}.
  • La expresió de return concluye la ejecución.

Invocación


                        let x = 8;
                        let z = productoDosNumeros(x, 3);
  • El código se ejecuta hasta que se la función se invoca.
  • Al invocar se pasan los valores de las variables.
  • Las variables dentro de la función, son locales.

Objetos

Una abstracción de los objetos en la vida real. Tienen propiedades y métodos.

Ejemplos: avión, horno de microondas. ¿Otros?

Se definan y crean objetos con una object literal 


                        let car = {type:"Fiat", model:"500", color:"white"};

Los pares nombre:valor son las propiedades.

Los métodos son acciones que se pueden ejecutar en los objetos y se guardan en propiedades mediante definiciones de funciones. 


                        let car = {
                            type:"Fiat", 
                            model:"500", 
                            color:"white",
                            descripcion: function(){
                                return this.type + ". Modelo: " + this.model + 
                                ". Color: " + this.color;
                            }
                        };

this hace referencia al objeto que es dueño de la función

Así accedemos a las propiedades y métodos de los objetos:


                        car.type;
                        car["model"];
                        car.descripcion();
                    

                        nombreObjeto.propiedad;
                        nombreObjeto["propiedad"];
                        nombreObjeto.metodo();

Arreglos

Los arreglos son variables especiales que nos permiten guardar múltiples valores. 


                        let arreglo = [item1, item2, ...];

Los elementos pueden ser de diferentes tipos y accedemos a cada uno utilizando índices. 


                        let frutas_favoritas = ["mango", "fresa", "guayaba", "mandarina"]; 
                        frutas_favoritas[0] // "mango"
                        frutas_favoritas[2] = "maracuya"
                        frutas_favoritas[frutas_favoritas.legth - 1]

Arreglos y Objetos

  • Los arreglos usan índices numéricos
  • Los objetos usan índices con nombres

Si quieres acceder a los elementos mediante strings, utiliza objetos.

Si quieres acceder a los elementos mediante números, utiliza arreglos.

typeof de un arreglo regresa object. Podemos utilizar la función Array.isArray() en nuevos navegadores.

Algunos de sus métodos:

  • Imprimir: toString() - join()
  • Quitar/añadir 1 elemento: pop() - push() - shift() - unshift(item)
  • Quitar/añadir varios elementos: splice(donde, cuantos, [items])
  • Unión: merge(arr1, arr2, ..)
  • Obtener varios elementos: slice(index, [index2])
  • Ordernar alfabéticamente: sort() - reverse()
  • Ordernar numérico: utilizamos una función de comparación
    • 
                                sort(function(a, b){return a - b})
  • No hay un método para obtener min y max. ¿Cómo podemos hacerlo?
  • Para ordenar arreglos de objetos de acuerdo a una propiedad:
    • 
                            sort(function(a, b){return a.propiedad - b.propiedad})

Los métodos de iteración operan sobre cada uno de los elementos de un arreglo.

  • forEach(). Llama a una función por cada elemento del arreglo. Recibe el valor, el índice y el arreglo.
  • map(). Crea un nuevo arreglo al llamar la función en cada elemento. Recibe: valor, índice y arreglo.
  • filter(). Crea un nuevo arreglo con los elementos que pasan una comparación. Recibe: valor, índice y arreglo.
  • reduce(). Ejecuta una función para reducir los elementos a un solo valor, izq a der. Recibe: total (valor anterior o inicial), valor, índice y arreglo. reduceRight() der a izq
  • every() Regresa un boolean que indica si todos los elementos pasan una comparación.
  • some() Regresa un boolean que indica si algunos de los elementos pasan una comparación
  • indexOf() - lastindexOf(). La primera y última ocurrencia de un valor en un arreglo.
  • find() - findIndex(). El primer valor y el primer índice que pasan una comparación.
