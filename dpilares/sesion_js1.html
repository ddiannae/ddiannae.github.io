En general, un programa es una lista de instrucciones ejecutadas por la computadora.

En un lenguaje de programació, las instrucciones se llaman declaraciones.

Las declaraciones se componen de: valores, operadores, expresiones, palabras reservadas y comentarios.

Sintaxis: Las reglas que componen un lenguaje. Describen las declaraciones válidas.

Valores

Hay dos tipos:

  • Literales. Valores fijos

                        /* Números */
                        10
                        8.340
                        /* Strings */
                        "Hola, con comillas dobles"
                        'Hola, con comillas simples'
  • Literales. Valores fijos
  • Variables. Guardan datos que pueden cambiar

                        var x; // Declaración
                        var a,b;
                        a = 10; // Asignación
                        b = 35;
                        x = a * b;

¿Qué identificadores de variables son válidos?

Antes de 2015, para declarar variables sólamente se usaba var, ahora se utiliza let y const

La diferencia entre var y let es su scope, dónde están disponibles las variables para ser usadas.

Por ahora: utilizar const si es una variable que no va a modificarse, y let en otros casos.
