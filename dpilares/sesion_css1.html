CSS

Significa Cascading Style Sheets. Describe cómo los elementos de HTML van a ser desplegados en la pantalla.

HTML describe el contenido. CSS fue creado por el World Wide Web Consortium (W3C) para evitar añadir tags para especificar diseño.

Sintaxis

¿Qué hace esta regla de CSS?


                        p {
                          color: red;
                          text-align: center;
                        }

Selectores de CSS

Sirven para seleccionar a qué elementos de HTML va a aplicarse el estilo. Diferentes tipos de selectores

  • Selectores simples: seleccionan elementos basados en nombre, id, clase.
  • Selectores combinados: seleccionan elementos basados relaciones entre ellos.
  • Selectores de pseudo-clase: seleccionan elementos basados en cierto estado.
  • Selectores de pseudo-elementos: seleccionan una parte de algún elemento.
  • Selectores de atributos: seleccionan elementos basados en un atributo o en el valor de algún atributo.

Height y Width

  • auto - default. El navegador lo calcula
  • tamaño - en px, cm, etc.
  • % - porcentaje del bloque que lo contiene
  • initial - lo coloca en su valor default
  • inherit - hereda width/height de su elemento padre

                        div.hw{
                            height: 200px;
                            width: 50%;
                            background-color: powderblue;
                        }

Recordemos que height y width no incluyen margin, padding y border por default.

max-width y max-height

Este div tiene un max-width de 500px. 

                        div.maxw {
                            max-width: 500px;
                            height: 100px;
                            background-color: powderblue;
                        }

También podemos modificar el min-height y min-width.

Display

Elementos que se despliegan como bloque: toman todo el ancho que pueden. display:block

  • <div>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Hola

Elementos que se despliegan en línea: solamente ocupan el ancho que requieren. display:inline

  • <span>
  • <a>
  • <img>

Podemos poner un span: Hola, dentro de un párrafo.


                        <p>Podemos poner un span: <span class="salmonback">Hola</span>, 
                        dentro de un párrafo.</p>

Position

Hay 5 diferentes valores:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Y después modificamos las propiedades de top, bottom, left y right.

Static. Valor por default. Flujo normal de la página.

div con position:static


                        div.st {
                            position: static;
                            border: 2px solid salmon;
                        }

Relative. Relativo a su posición original.

div con position:relative


                        div.rl {
                            position: relative;
                            left: 250px;
                            border: 2px solid salmon;
                        }

Fixed. Relativo al viewport.

div con position:fixed


                        div.fx {
                            position: fixed;
                            left: 400px;
                            top: -30px;
                        }

Absolute. Relativo al ancestro más cercano con posición diferente a static.

Yo tengo posición relativa

div con position:absolute


                        div.abs {
                            position: absolute;
                            right: 100px;
                            bottom: -30px;
