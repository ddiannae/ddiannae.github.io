Semana 2: La Web

La Web

¿Cómo explicarías qué es la web (World Wide Web)?

La WWW o Web no es lo mismo que el Internet

Web: Es un sistema para distribuir documentos. Es la arquitectura lógica de la información.

¿Qué tipo de documentos?

HTML Hypertext Markup Language

Tim Bernes-Lee en 1989, con la ayuda de Robert Cailliau mientras trabajaban en el CERN en Ginebra, Suiza

La WWW o Web no es lo mismo que el Internet

Internet: Es una red física que conecta dispositivos.

Inició como una conexión entre computadoras de varias universidades de California en 1969. Se llamaba ARPANET.

Transmite diversos servicios a través de varios protocolos

Protocolos

Es un sistema de reglas para que dos dispositivos puedan transmitir información dentro de un sistema de comunicación.

  • Cliente: Dispositivos que se conectan a internet.
  • Servidor: Almacenan páginas web o aplicaciones.

La información se transmite a través de paquetes.

Ustedes van a convertirse en desarrolladoras Web, ¿esto qué significa?

Front-end

Se ejecuta en el lado del cliente, es la interfaz de usuario. Recoge los datos del usuario y se muestra en un navegador web.

Back-end

Se ejecuta en el servidor. Procesa la información que recoge del Front-end y envía una respuesta.

¿Qué pasa cuando accedemos a una página Web?

  1. El navegador va al servidor DNS y encuentra la dirección IP del servidor.
  2. El navegador envía un mensaje de petición HTTP al servidor, pidiéndole que envíe una copia de la página web para el cliente. Internet y protocolo TCP/IP
  3. Si se aprueba la soliciud, el servidor comenzará a enviar los archivos de la página web al navegador como paquetes.
  4. El navegador reúne los pequeños trozos, forma un sitio web completo y te lo muestra.

Git

Es un sistema de control de versiones distribuido.

Hay tres estados para los archivos

  • Confirmado: Los datos están almacenados de manera segura en tu base de datos local.
  • Modificado: Se ha modificado el archivo pero no se ha confirmado en la base de datos local.
  • Preparado: Se ha marcado un archivo modificado en su versión actual para que vaya en tu próxima confirmación.

Hay tres secciones principales para un proyecto de Git

El servidor para repositorios remotos más popular: GitHub.
